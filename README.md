# FlutterEffect 🦋

## Problem
Küresel su krizi her geçen gün büyürken, insanlar günlük tüketim alışkanlıklarının (örneğin sabah içilen bir fincan kahvenin) dünyanın öbür ucundaki su kaynaklarını ve ekosistemleri nasıl tükettiğinin farkında değil. İnsanların tüketim tercihlerinin yarattığı bu yıkıcı "Kelebek Etkisi"ni ve görünmez su israfını görünür kılma problemini çözüyoruz.

## Çözüm
FlutterEffect, kullanıcıların tüketim alışkanlıklarının küresel su kaynaklarına olan etkisini simüle eden yapay zeka destekli bir platformdur. Sistemin kalbindeki **Gemini AI**, kullanıcının girdiği ürün verisini analiz ederek anlık su ayak izini ve bölgesel kuraklık riskini hesaplar. Sadece veriyi göstermekle kalmaz; kullanıcıya anında çevre dostu alternatif ürünler ve kişiselleştirilmiş su tasarrufu senaryoları sunarak değişime teşvik eder.

## Kullanılan Teknolojiler
- HTML5 & CSS3 (Modern, Responsive Arayüz)
- Vanilla JavaScript (Asenkron API Yönetimi)
- Google Gemini API (Yapay Zeka Analiz ve Öneri Motoru)
- Netlify (Statik Web Hosting)
- Cursor AI (Geliştirme Ortamı)

## Nasıl Çalıştırılır?
Bu proje Netlify uyumlu statik bir web sitesi olarak optimize edilmiştir. Herhangi bir yerel sunucu (localhost) kurulumuna veya backend bağımlılığına ihtiyaç duymaz. 
1. Proje dosyalarını bilgisayarınıza indirin.
2. Klasör içindeki `index.html` dosyasını doğrudan herhangi bir web tarayıcısında (Chrome, Safari vb.) açın.
3. Uygulama anında çalışmaya başlayacak ve Gemini API arka planda JS üzerinden tetiklenecektir. Alternatif olarak yukarıdaki "Canlı Demo" linki üzerinden kurulumsuz erişebilirsiniz.
